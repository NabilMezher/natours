# natours

Natours is a fictional tour booking platform that allows users to discover and book exciting tours around the world. The application showcases various concepts and features of full-stack web development using Node.js, Express.js, and MongoDB.

# Features
User authentication: Sign up, login, and logout functionality.
Tour browsing: Explore available tours, read details, and view photos.
Tour booking: Reserve tours, make payments, and receive booking confirmations.
User reviews: Read and submit reviews for tours.
User profile management: Update personal details, change passwords, and manage bookings.
Admin features: Create, update, and delete tours, manage user accounts, and handle bookings.

# Technologies Used
Node.js: JavaScript runtime environment.
Express.js: Web application framework for Node.js.
MongoDB: NoSQL database for storing application data.
Mongoose: Object Data Modeling (ODM) library for MongoDB.
Pug: Template engine for rendering dynamic views.
